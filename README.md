# OccDet-Calib

Occlusion-Calibrated Object Detection:
Diagnosing and Mitigating Calibration Failure Under Partial Visibility.

## Core detectors
- YOLOv8m
- FCOS R50-FPN
- Deformable DETR R50

## Core goals
- Build OccDet-Calib benchmark
- Study calibration vs occlusion
- Test Dangerous Zone hypothesis
- Compare post-hoc calibration baselines
- Implement OC-TS and abstention

## Repo structure
- `src/` code
- `configs/` experiment configs
- `data/` local placeholders only
- `outputs/` ignored large artifacts
- `paper/` writing assets
